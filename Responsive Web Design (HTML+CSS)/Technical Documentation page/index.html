Introduction

Python is a high-level, interpreted programming language known for its readability.

It was created by Guido van Rossum and first released in 1991.

Python is widely used in web development, data science, and automation.

Variables

In Python, variables store data without needing explicit declaration.

For example: x = 10 assigns the integer 10 to x.

You can also use name = "Alice" for strings.

Functions

Functions in Python are defined with the def keyword.

Example: def greet(name): return f"Hello, {name}"

Call it with print(greet("Bob")) to output "Hello, Bob".

Loops

Python uses loops to repeat tasks efficiently.

A for loop example: for i in range(5): print(i)

  • Prints numbers 0 to 4.
  • Loops over lists: for x in [1, 2, 3]: print(x)
  • While loops run until a condition is false.
  • Use break to exit early.
  • Use continue to skip an iteration.
